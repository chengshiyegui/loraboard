Setup

The LoRa endpoint talks to a LoRa gateway which in turns talks to the Loriot service. The Lioriot.io service has been configured to publish its received data to the PubNub service. This page subscribes to the corresponding PubNub channel to get the realtime LoRa data, decodes it and displays live charts.

Technical details

This page relies on PubNub js library to gather realtime data and on the C3.js library for charts.

Just open the browser console to see the incoming LoRa frames (F12 & console tab in chrome).
